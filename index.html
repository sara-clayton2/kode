HTML

Web pages are files that are viewed using a browser. They are written in a language called HTML. HTML is made up of a set of elements that are used as instructions to tell the browser what should appear on a web page and how it should be structured.

Understanding Common HTML terms

Elements
Elements are designators that define the structure and contents of the objects within the web page. Some of the most commonly used elements include different levels of headings as h1 through h6 and paragraphs identified as the p element. Many more styles include a , div , span , strong and em.
Elements can be identified by the use of the less than < and greater than brackets > surrounding the element name.

Tags
The use of less than < and greater than > angle brackets surrounding an element creates something called a tag. Tags occur in pairs of opening and closing tags. An opening tag marks the beginning of an element. It consists of a less than sign followed by an elementâ€™s name, and then ends with a greater than sign.
A closing tag marks the end of an element. It consists of a less than sign followed by a forward slash / and the elements name, ending with a greater than sign .The content that falls between the opening and closing tags is the content of that element.

Attributes
Attributes are properties used to provide additional information about an element. They are used to provide additional information about an element.
The most common attributes include the id attribute which identifies an element, the class attribute which classifies an element; the src attribute which identifies a source for embeded content.
Attributes are defined within the greater than or less than tags after an elements name. These are formatted by the attribute name followed by an equals sign and then a quoted attribute value.

HTML is like the skeleton of your website. It provides the structure, and starts with a body element thats divided into three sections: header, main and footer.
Click on the tabs to learn more!

< header >

The entire body of your HTML code starts with a header element. It represents a container for introductory content or a set of navigational links.

< main >

The main part of your code is whats going to contain all important content that is related to the topic of your website. This is where you can add text, links and images.

< footer >

The footer is the end of your website. This is where you can add copyright information, contact information and extra navigation links.

CSS describes how HTML elements are displayed. It helps make your website stylized and dressed up. With CSS you can add fonts, colors, background images, widths, heights and so much more. So have fun with it!
